DATA CALCULATION LOGIC:
=======================

For Enfra Regional Graph:
-------------------------
1. Filter regionData.enfra for specific regions
2. Extract counts for:
   - Jacobabad: regionData.enfra['Jacobabad'] or 0
   - Larkana: regionData.enfra['Larkana'] or 0
   - Sukkur: regionData.enfra['Sukkur'] or 0
3. Display as bar chart
4. Tooltip shows: "Region: Count (X% of Enfra)"

For SMS LD Regional Graph:
---------------------------
1. Filter regionData.smsLd for specific regions
2. Extract counts for:
   - Jacobabad: regionData.smsLd['Jacobabad'] or 0

   - Sukkur: regionData.smsLd['Sukkur'] or 0
3. Display as bar chart
4. Tooltip shows: "Region: Count (X% of SMS LD)"
